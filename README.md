# Inventory Client

## System Requirements
- **Java**: Version 25 or later
- **Operating System**: Windows 10/11
- **Backend**: Spring Boot API running on port 8080

## Installation
1. Extract the distribution ZIP file
2. Run `scripts/install.bat` to verify Java installation
3. Ensure the backend API server is running
4. Double-click `student-management-client.exe` to start the application

## Configuration
- Update `config/app-config.properties` to change API endpoints
- Application logs are stored in `logs/student-management-client.log`
